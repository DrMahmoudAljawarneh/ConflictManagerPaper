Introduction

UbiREAL is a smartspace simulator written in Java. It consists in the minimum required basic simulation functions, and includes only the 3D space simulator: the other entities can be implemented as plugin.

Users can freely create, insert and set some new entities in the simulator.

Starting and command line options

UbiREAL needs to be launched in the Java environment with the JOGL library installed.

To start the application, make sure the JOGL .jar files (jogl.jar, glue-gen-rt.jar), the UbiREAL .jar files (cybergarage.jar, ubireal2.jar) as well as the user-defined .jar files are set to the classpath. It is required to add the directory in which the JOGL .dll files are located in the java.library.path.

It is recommended to set the heap memory to more than 1GB. The main class of UbiREAL is org.itolab.nsibata.ubireal2.main.UbiReal2.

The command line options are described in the table below:

Command Line Option Result
--cachedir [directory] set the cache directory
--project [project file] set the project file

An example of the command line to input in the Windows Command Prompt is given below:

java -cp ./jogl/jogl.jar;./jogl/gluegen-rt.jar;cybergarage.jar;ubireal2.jar;. -Djava.library.path=./dll -Xmx1024m org.itolab.nsibata.ubireal2.main.UbiReal2 --cachedir c:/tmp --project test.ur2

Main Screen

On the main screen, the different parts from up to bottom are respectively the pull-down menu, the 3D visualizer, the command panel and the status bar. The standard messages as well as the error messages are displayed in the command panel. Those messages are also automatically saved in the ubireal.log file.

Main Screen Features

The camera of the main screen can be moved using different keyboard commands, mentioned in the table below.

Key Commands Action
Directional Keys Camera Movement
Mousewheel Zoom in - out
Ctrl + Directional Keys Fast Camera Movement
Ctrl + Left Button Drag Camera Panning
F5 Key Simulation Start
End Simulation Reset
F4 Key Plugin Control Panel
Ctrl + S Save Project
F1 Key Help Page


File Menu

In the file menu, users can load and save the project file. The status of each plugin and the camera position are recorded in the project file.


Simulation Menu

Users can start, pause or reset the simulation using the simulation menu.


Tools Menu

Through the Plugin Configuration window, the plugins and the entities can be set (Fig. 2). Selecting a plugin or an entity on the left panel activates the Config button that allow to display each configuration window.

Moreover, some entities can be moved through their respective Plugin Configuration window. For each of those entities, a click on the Move button will display its EntityMover window (fig. 3). Similarly, the Help button links to a webpage for some plugins and entities.

Fig.2 Plugin Configuration Window

Dragging the Left button of the mouse in each of the window rectangles will move the entity in the visualizer. The Undo button allows to undo the last action, while with the Reset button will allow to set back to the initial status.


Fig.3 EntityMover Window

Help Menu

Via this menu, users can access the UbiREAL Homepage as well as the Help Page. The information about the application version can be also displayed.


Simulator Configuration

The parameters of the Tool -> Simulator Config panel are presented in the table below.

Parameters Meaning
simulationSpeed simulation speed
visualUpdateFrequency refresh frequency
OpenGLFrameRate standard framerate
noVisual no display


About the cache file

Some of the operations are saved as cache files in the cache directory. Deleting them does not affect the application.
