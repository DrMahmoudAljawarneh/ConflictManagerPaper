Air Conditioner en

Introduction

This plugin sets an aircon in the 3D environment


Setting Parameters

Below are the setting parameters

Table 1 Setting Parameters
Parameters Meaning
model sets the appearance of the object using a value from 0 to 2
power integer variable to define whether the electricity is set or not
temperature sets the initial temperature
humidity sets the initial humidity
outputCapacity boolean variable to define whether heat is produced from the object
