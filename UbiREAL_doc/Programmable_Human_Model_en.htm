Introduction

ProgrammableHumanModel is model in which the human behavior can be described in as a program.

Setting parameters

The following parameters can be defined in the entity:


pictureFilename

  • set the avatar filename

height

  • set the height of the avatar (unit: m)

classname

  • set the name of the subclass of ProgrammableHumanModel.HumanProgram

Programming the human behavior

The human behavior is defined both by a graph drawn using the plugin and a code written in Java. The graph node can be named for its reuse in the program.

The main class of the human behavior program must be a subclass of the ProgrammableHumanModel.HumanProgram; the startForm and main methods being the required methods of the created class.

The startForm method returns the name of the graph node on which the avatar is initially located. The main method is run at first and set the behavior of the avatar. 

import org.itolab.nsibata.ubireal2.plugin.*;

public class PHMTest extends ProgrammableHumanModel.HumanProgram {
    public String startFrom() {
        return "0";
    }

    public void main() {
        for(;;) {
            goTo("1", 1, false);
            wait(1, false);
            goTo("0", 1, false);
            wait(1, false);
        }
    }
}
