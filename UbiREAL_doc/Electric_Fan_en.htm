Electric Fan en

Introduction

This plugin displays an electric fan in the 3D environment.

Setting Parameters

Table1 Setting Parameters
Parameters Meaning
model sets the appearance of the object using a value from 0 to 2
power boolean variable to define whether the electricity is set or not.
headAngle sets the initial angle the of the fan head
speed sets the initial rotation speed
oscillation boolean variable to define the initial oscillation status
updatePeriod set the update timing of the object
