Simple Human Model en

The SimpleHumanModel is simple human model similar to the one in UbiREAL Version 1. Below are the 4 setting parameters:


Contents

  • 1 pictureFilename
  • 2 height
  • 3 zpos
  • 4 speed

pictureFilename

  • set the avatar file

height

  • set the height of the avatar

zpos

  • set the leg length of the avatar

speed

  • set the walking speed of the avatar
